## 🔧 优化改进

### 面试评分系统
- 面试结束由 DeepSeek AI 评估全部问答，给出综合得分+四维评分+逐题点评
- API 不可用时降级为规则引擎（根据回答长度/量化数据/跳题数动态算分）
- 回答不足或提前结束面试时自动给低分，避免不合理高分
- 面试结束显示"AI正在评估"等待动画

### 面试体验
- 已问题目传给AI避免重复出题
- 语音输入改为引导使用微信桌面版全局语音识别（Ctrl+Win / Ctrl+Shift+Win）
- 倒计时延长至3分钟/题
- 追问次数根据回答质量动态调整（1-3轮）

### 简历分析增强
- 新增岗位描述（JD）输入框，AI根据JD定向匹配分析
- 支持PDF/Word文件上传解析
- 分析完成后输入区自动折叠

### 报告优化
- 逐题回顾加入 STAR 框架说明
- 截图时自动展开所有折叠面板，确保内容完整
- 新增文本导出功能（完整题目+回答+点评）
